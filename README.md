# EventSphere - Event Booking & Management System

This is a Next.js project set up with React, TypeScript, and Tailwind CSS. It serves as the foundation for the EventSphere web application.

## Technologies Used
- **Next.js** (App Router)
- **React**
- **TypeScript**
- **Tailwind CSS**
- **ESLint** for static code analysis

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
